Frequently Asked Questions

Does ngrok work behind an HTTP Proxy?

Yes. You may specify an HTTP proxy to connect through in ngrok's configuration file or by using the standard unix environment variable http_proxy. Consult the documentation on running ngrok through an HTTP proxy for more details.

I use Apache with virtual hosts for development. Can I use ngrok to expose one of my sites?

Yes. Use Apache's ServerAlias directive to properly route tunneled ngrok requests to your site. 

<VirtualHost *:80>
    ServerName  example.dev
    ServerAlias example.ngrok.com
    ...
<VirtualHost>
Then invoke ngrok requesting the proper subdomain to tunnel requests from example.ngrok.com to your example.dev site: 

ngrok -subdomain example 80

I use nginx with virtual hosts for development. Can I use ngrok to expose one of my sites?

Yes. Use multiple arguments to nginx's server_name directive to properly route tunneled ngrok requests to your site. 

server {
    listen       80;
    server_name  example.ngrok.com example.dev;
    ...
}
Then invoke ngrok requesting the proper subdomain to tunnel requests from example.ngrok.com to your example.dev site: 

ngrok -subdomain example 80

Can I tunnel a Wordpress site over ngrok? When I try, all of the links are broken.

Yes. You need to add one of the following plugins to force Wordpress to issue relative URLs.

  • https://github.com/optimizamx/odt-relative-urls
  • http://wordpress.org/plugins/relative-url/
  • http://wordpress.org/plugins/root-relative-urls/

What information is logged or stored about my tunnels on ngrok.com?

ngrok does not log or store any data transmitted through your tunneled connections. ngrok.com does log some information about the connections which are used for debugging purposes and metrics like the name of the tunnel and the duration of connections. You may audit the source code for the ngrok server to see exactly what is logged on the production server.

Can I run my own ngrok server?

Yes. ngrok's server code is open-source and available on the ngrok github project Consult the documentation for running your own ngrokd server

Does ngrok tunnel websocket connections?

Yes. However, there is currently no support for introspecting them.
